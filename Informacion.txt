
--- Exportar o Importar Librerias Necesarias ---

        Para sacar la lista de las librerias se usa:

        - pip freeze > "requirements.txt"

        Para instalar la lista de las librerias que se requiren:

        - pip install -r "requirements.txt"

--- Para Correr El Proyecto --- 

        Modo Produccion:

         - flask --app 'nombre de la app' run

        Modo Desarollador:

        - flask --app 'nombre de la app' --debug run
